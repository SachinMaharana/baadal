# nimbus

```
nimbus 0.1.0
nimbus cli interacts with cloudflare for managing dns records

USAGE:
    nimbus --cloudflare_token <cloudflare-token> <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -c, --cloudflare_token <cloudflare-token>    or export env variable [env: CLOUDFLARE_TOKEN]

SUBCOMMANDS:
    dns     
    help    Prints this message or the help of the given subcommand(s)

```
